## Description
A vermillion fluid with gray oddiments. It smells like honey and herbs, but tastes like beer and mead.

## Effects
Allows anyone proficient in a class of tools to complete one task with the tools in half the normal time. If the imbiber is interrupted, the effect is lost.

## History
This potion was created by the artificer Weebly to improve concentration while working on a project.
