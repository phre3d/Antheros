# Description
## Description 
An azure mixture. It smells like peach, but tastes like raspberries and blueberries.

## Effects 
When you drink this potion, your melee attacks deal an extra 1d8 damage. This effect ends early if you fail to take an attack action against a creature for more than 30 seconds.
