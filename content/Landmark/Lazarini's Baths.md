# Description
## A nice place to get clean 

## Description 
These private baths serve [[Organization/Criminal/The Black Hand|The Black Hand]] as a [[landmark/Message Drop|Message Drop]]. Messages or small packages are kept by the day clerk behind the counter, marked "For Jonathon". "Jonathon" is a code word, and the clerk will give the message or package to anyone claiming that name. Several of [[Organization/Criminal/The Black Hand|The Black Hand]] sworn swords come here to cool off after workouts.
